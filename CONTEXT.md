# Multi-Agent Memory

本上下文描述多智能体低开销协作原型中的核心领域语言，用于统一系统设计、代码和实验报告中的概念。

## Language

**结构化协作协议**:
Agent 之间传递动作、参数、结果、能力和状态引用的紧凑消息格式。
_Avoid_: 长文本透传, 自然语言协议

**状态包**:
由任务文本或中间结果生成的非文本向量表示，可被 Agent 直接传递和复用。
_Avoid_: 文本摘要, 普通上下文

**记忆单元**:
共享记忆库中的可检索记录，包含来源 Agent、任务主题、摘要、标签、证据和向量表示。
_Avoid_: 日志行, 缓存项

**记忆生命周期**:
记忆单元的类型、版本、状态、命中次数、最近命中时间和置信度，用于描述记忆是否仍然活跃、可靠和常被复用。
_Avoid_: 普通创建时间, 文件时间戳

**动态记忆链接**:
记忆单元之间的父子关系、复用关系和演化原因，用于描述新记忆从哪些历史经验发展而来。
_Avoid_: 标签相似, 普通引用

**组织记忆**:
从多智能体协作轨迹中抽取出的可迁移分工模式，包含角色、队形、触发条件和策略参数。
_Avoid_: 任务日志, 普通经验摘要

**围堵任务**:
两个追捕者通过分工夹击捕获一个逃逸者的连续控制任务，用于验证组织记忆能否迁移给新的 Agent。
_Avoid_: 普通导航样例, 单 Agent 追逐

**组织记忆基准套件**:
一组用于评估组织记忆迁移能力的多智能体场景，覆盖追捕围堵、协同导航覆盖和接力运输。
_Avoid_: 单一演示, 孤立案例

**协作拓扑**:
组织记忆中描述角色之间依赖关系的结构，例如左右夹击者互相约束、中心覆盖者连接两侧覆盖者。
_Avoid_: 调用顺序, 消息链

**MADDPG-like 压缩策略**:
使用集中训练信号抽取共享 actor 基和角色 adapter，并在执行时由每个 Agent 分散使用的低维策略表示。
_Avoid_: 完整轨迹回放, 大模型提示词缓存

**连续任务组**:
用于验证跨任务记忆复用的一组相关任务，后续任务应能命中并复用前序任务沉淀的记忆。
_Avoid_: 单次任务, 无关样例

**纯文本协作模式**:
Agent 之间使用自然语言长上下文传递协作信息的基线模式。
_Avoid_: baseline, text-only

**结构化协议协作模式**:
Agent 之间使用结构化协作协议、状态包和记忆单元完成任务的实验模式。
_Avoid_: optimized mode, JSON mode
